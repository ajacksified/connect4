Connect 4
=========

Install and play:

1. npm install
2. chmod +x bin.js
3. ./bin.js

How to play:

Enter in a row to drop in (1-7). Take turns. Whoever gets 4 in a row
(horizontally, vertically, or diagonally) wins.
